File Path

Their are two types of file path

  1. Absolute
  2. Relative

Absolute File Path

It's a file path that is relative to the root of the computer.

Example:

  • C Drive in Windows
  • Macintosh HD in Mac

Relative File Path

A relative file path is relative to something.

Display Property

Their are basically 3 types of display properties:

  • block:

    It takes up the entire full width. Now, this means that if you have another element, then it's going below it, not on the same line.

  • inline:

    If we have one element, then add another element, then it's going to go on the same line until it fit the width of the webpage. You can't set the size of the element.

  • inline-block:

    It allows us to set the size of the element and the elements go on the same line.

  • none:

    It hides the element from the webpage.
